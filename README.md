# stm32_cubeIDE
STM32_CUBEIDE 

STM32 BLUE PILL PROGRAM USING STM32CUBEIDE



    ST LINK V2 TO BLUE PILL
    
| ST-LINK Pin        | Blue Pill Pin |
| ------------------ | ------------- |
| 3.3V   (7 OR 8)    | 3.3V          |
| GND    (5 OR 6)    | GND           |
| SWDIO  (4)         | SWDIO         |
| SWCLK  (2)         | SWCLK         |
